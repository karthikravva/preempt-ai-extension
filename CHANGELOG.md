# Changelog

All notable changes to the Preempt AI Browser Extension will be documented in this file.

## [1.0.0] - 2024-12-09

### Added
- Initial release
- Prompt injection detection and blocking
- Jailbreak attempt prevention
- PII detection and protection (SSN, credit cards, emails, phone numbers)
- Support for major AI platforms:
  - ChatGPT (chat.openai.com, chatgpt.com)
  - Claude (claude.ai)
  - Google Gemini (gemini.google.com)
  - Microsoft Copilot (copilot.microsoft.com)
  - Perplexity (perplexity.ai)
  - And 30+ more AI platforms
- Quick scan popup for manual prompt checking
- Real-time protection toggle
- API status indicator
- Copy protected prompt functionality

### Security
- All API communication over HTTPS
- No data storage or logging
- Minimal permissions requested
